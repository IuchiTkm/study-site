Git
コミット
変更内容をリポジトリに記録するスナップショット。
ブランチ
コミット履歴の分岐。機能開発や修正を独立して進めるために使う。
リポジトリ
変更履歴を管理するためのデータの保管場所。
Linux
パーミッション
ファイルやディレクトリに対する読み・書き・実行の権限設定。
プロセス
実行中のプログラムの単位。PID（プロセスID）で識別される。
Git
コマンド 説明
git clone <url> リモートリポジトリのブランチを複製してカレントディレクトリに保存する
git init ローカルリポジトリを新規作成する
git remote リモートリポジトリの操作を行う
git remote add <name> <url> ローカルリポジトリにリモートリポジトリを追加する
git remote rm <name> リモートリポジトリを削除する
git remote set-url <name> <url> リモートリポジトリのURLを変更する
git remote rename <old> <new> リモートリポジトリ名を変更する
git remote show <name> リモートリポジトリの詳細情報を表示する
git branch ローカルリポジトリのブランチを操作する
git checkout <branch> ブランチの作成や切り替え、コミット切り替えを行う
git switch <branch> ブランチの切り替えや作成を行う
git restore <file> ブランチの作業内容を元に戻す
git add <file> ファイルをインデックスに追加する（ステージング）
git status ステージングされたファイルの一覧を表示する
git rm <file> 削除したファイルを指定してステージングする
git commit -m "<msg>" インデックスのファイルをリポジトリに記録する
git push ローカルリポジトリの変更内容をリモートに送信する
git pull リモートリポジトリの内容を取得して取り込む
git fetch ローカルとリモートリポジトリを同期して変更情報を取得する
git merge <branch> 別ブランチのコミットを作業中のブランチにマージする
git cherry-pick <hash> 指定したコミットを作業中のブランチにマージする
git rebase <branch> コミットをブランチに適用して再編成する
git show <hash> 指定したコミットの履歴を参照する
git log コミットの履歴と詳細を表示する
git reflog コミットの履歴を表示する（削除分も含む）
git reset インデックスを削除またはコミット前の状態に戻す
git revert <hash> 指定したコミットの打ち消しを行う
git stash 作業中ブランチの内容を退避する
git diff コミット同士や作業内容の差分を表示する
Docker
コマンド 説明
docker image
docker image pull <image> dockerhubからイメージを取得する
docker image ls イメージの一覧を表示する
docker image rm <image> イメージを削除する
docker image prune コンテナから参照されていない未使用のイメージを全て削除する
docker image inspect <image> イメージの詳細情報を表示する
docker image build <path> 指定したDockerfileからイメージをビルドする
docker image tag <src> <target> イメージのタグ名を変更する
docker container
docker container ls コンテナの一覧を表示する
docker container rm <container> コンテナを削除する
docker container prune 実行されていないコンテナを全て削除する
docker container run <image> 新規コンテナをイメージから生成する
docker container start <container> コンテナを起動する
docker container stop <container> コンテナを停止する
docker container restart <container> コンテナを再起動する
docker container exec <container> <cmd> 実行中のコンテナに対してコマンドを実行する
docker container cp <src> <dest> コンテナとローカル間のファイルのコピーを行う
docker container logs <container> コンテナのログを表示する
docker container stats コンテナのリソース情報を動的に表示する
docker container top <container> コンテナのプロセス情報を表示する
docker container diff <container> コンテナ内のファイルやディレクトリの変更箇所を確認する
docker container inspect <container> コンテナの詳細情報を表示する
docker network
docker network create <name> ネットワークを作成する
docker network connect <network> <container> コンテナとネットワークを接続する
docker network disconnect <network> <container> コンテナとネットワークを切断する
docker network ls ネットワークの一覧を表示する
docker network inspect <network> ネットワークの詳細情報を表示する
docker network rm <network> ネットワークを削除する
docker network prune 参照されていないネットワークを削除する
docker volume
docker volume create <name> ボリュームを作成する
docker volume ls ボリュームの一覧を表示する
docker volume inspect <name> ボリュームの詳細情報を表示する
docker volume rm <name> ボリュームを削除する
docker volume prune 参照されていないボリュームを削除する
その他
docker search <keyword> dockerhub上のイメージを検索する
docker save -o <file> <image> コンテナのベースイメージとレイヤーの情報を全てtarファイルに出力する
docker export -o <file> <container> コンテナの最終的な内部構成をtarファイルに出力する
docker load -i <file> saveコマンドで保存したファイルを取り込む
docker import <file> exportで出力したファイルを取り込む
Linux
コマンド 説明
cd <dir> ディレクトリを移動する
chmod <mode> <file> ファイルのパーミッションを変更する
grep <pattern> <file> ファイル内からパターンを検索する
ls -la ファイル一覧を詳細表示（隠しファイル含む）
pwd 現在のディレクトリパスを表示する
